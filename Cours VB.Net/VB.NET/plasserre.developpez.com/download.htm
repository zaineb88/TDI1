 LDF 1.5  (2003): Ancienne Version: Télécharger l'ensemble du pack 

Première installation:

1-Télécharger  LDFinstall.exe  (2,4 Mo) en cliquant sur le lien suivant: Télécharger le pack
2-Cliquer sur LDFinstall.exe, puis sur le bouton "Décompresser": 3 fichiers apparaîtront dans le répertoire TEMP sous le répertoire Windows. 

3-Cliquer sur SetUp dans ce répertoire TEMP , l'installation démarrera.

4-Pour exécuter LDF: Bouton "Démarrer" puis "Programmes" puis "LDF" puis "LDF"

LDF 1.7.1 (21/05/2007) Ancienne Version: Télécharger uniquement la mise à jour 

LDF  doit être déjà installé, télécharger uniquement l'exécutable. Les données seront conservées.

Formats : fichier: MAJ17.EXE           

1-Télécharger MAJ.exe  en cliquant sur le lien suivant: Télécharger MAJ17

2-Double-cliquer sur MAJ.exe, puis sur le bouton "Ok": Le fichier LDF.exe  est décompressé dans le répertoire habituel de LDF (c:\Program Files\LDF\)
Avec Windows Vista il faut mettre la mise à jour dans 'Bureau- Ordinateur-Disquelocal (c:)-LDF'

3-Lancer LDF comme d'habitude.
