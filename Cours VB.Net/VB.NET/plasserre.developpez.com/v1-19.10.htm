Réponses:

 

10- Les variables 'Object'.

 

10.1 Créer une variable nommée 'myObjet' de type 'Object'. Y mettre "Bonjour" . 

 

Dim myObjet As New Object

myObjet="Bonjour"

 

Quel est le type de ce qu'il y a dans myObjet? Comment afficher ce type dans une messageBox?

 

MsgBox.Show(myObjet.GetType.ToString)

Affiche: System.String

 

Quelles sont les propriétés de myObjet? 

 

Dans le code il suffit de taper "myObjet.", on voit apparaître dans la liste: Equals, GetHashCode, GetType, ReferenceEquals, ToString. Ce sont les propriétés des Object.

 

Quelles sont les propriétés d'une String?

 

Clone ..Compare.. Contains..Length...

 

Comment utiliser les propriétés des strings sur MyObjet, donner un exemple( Afficher dans une MessageBox la longueur de la chaîne).?

Il faut d'abord le transformer l'objet en String à l'aide de CType.

MsgBox(CType(myObjet, String).Length.ToString)
