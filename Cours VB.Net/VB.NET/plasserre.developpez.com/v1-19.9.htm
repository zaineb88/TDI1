9.1 Déclarer une variable Integer nommée 'myNumber'. L'initialiser  avec une valeur quelconque.

Tester si la valeur est comprise entre 2 et 5 compris, si oui afficher "entre 2 et 5".

 

Vous avez probablement utilisé un And, utiliser maintenant un Or (plus un Not) pour faire la même chose. 

 

9.2 Déclarer une variable nommée 'myBol' devant contenir True ou False.

Initialiser avec True.

Inverser la valeur.

Tester si la valeur est égale à False.

 

 

Réponses:

 

9- Logique.

 

9.1 Déclarer une variable Integer nommée myNumber. L'initialiser  avec une valeur quelconque.

Tester si la valeur est comprise entre 2 et 5 compris, si oui afficher "entre 2 et 5".

 

 

    Dim myNumber As Integer =12

    If myNumber>=2 And myNumber<=5 Then

            MsgBox ("entre 2 et 5")

    End If

Vous avez probablement utilisé un And, utiliser maintenant un Or (plus un Not) pour faire la même chose. 

  If Not (myNumber<2 Or myNumber>5) Then...

 

9.2 Déclarer une variable nommée 'myBol' devant contenir True ou False.

Initialiser avec True.

Dim myBol As Boolean =True

Inverser la valeur.

myBol= Not myBol   

Tester si la valeur est égale à False.

If myBol = False Then

            MsgBox ("valeur=False")

End If
