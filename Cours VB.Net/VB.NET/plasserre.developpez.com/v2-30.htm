Les Couleurs  .

 

A- Couleur VB.

B- Couleur RVB.

C- Couleur du WEB.

 

Une couleur est représentée par 3 octets correspondent aux composants de couleur rouge, vert et bleu plus la composante alpha éventuellement qui indique la transparence. Chaque octet peut prendre la valeur 0 à 255 (ou 0 à FF en hexadécimal).

Si on utilise la notation hexadécimale, il faut mettre &H avant.

 

Exemple : valeur des 3 composantes couleurs et couleurs correspondantes:

  rouge   vert   bleu  
0 0 FF  
FF 0 0  
80 0 80  
0 0 0  
FF FF FF  

 

En plus, dans certains cas, il y a une composante alpha qui indique la transparence. 255 indique que la couleur est opaque, 1 à 254 indique que la couleur est transparente.

Il y a une Classe Color dans SystemDrawing. On peut instancier un Objet Color:

Dim myColor As Color

On peut voir ou modifier les composants de cette couleur avec:

myColor.A composante alpha

myColor.B composante bleue

myColor.R composante rouge

myColor.G composante verte

 

A- Couleur VB.

Le plus simple est, pour modifier la couleur d'un objet par du code, d'utiliser l'énumération VB qui contiennent le code d'une couleur toute faite (en RGB sans composante Alpha):

Dim myColor As Color =Color.Back,

Color.Fuchsia

Color.Chocolate

Color.Red ...

Elles font partie de  System.Drawing

En mode 'Design' dans VB, dans la fenêtre de propriétés des objets visuels, si on veut modifier une propriété comme BackColor, en cliquant sur '...', on a accès aux couleurs 'Web' qui correspondent en fait aux couleurs VB:

 

A- Couleur RVB.

Dim myColor As Color =Color.FromArgb(&H780000FF); 78 00 00 est la couleur FF la composante alpha. 

On peut aussi utiliser l'instruction VB RGB:

Dim red As Color = RGB(255, 0, 0)

C- Couleur Web.

Il y a longtemps quand on avait des moniteurs affichant 256 couleurs, il existait une liste nommé 'web-safe colors ' contenant 216 couleurs qui étaient des couleurs 'sures' c'est à dire affichable sans utilisation de tramage (le tramage étant l'affichage d'une couleur en juxtaposant des pixels de couleurs différentes pour se rapprocher de la couleur manquante); mais maintenant cela est obsolète avec les moniteurs actuels.

Il semble que les couleurs VB correspondent aux couleurs Web bien qu'il y aie plus de couleurs Web que de couleurs VB.

A vérifier.

Dans une feuille de style, d'un document HTML ou XML,  on peut indiquer une couleur:

color : red;                    'nom de couleur en anglais voir la liste des couleurs X11 pour le Web
color : #fd4a88;                'en hexadécimal
color : rgb(125,32,98);         'en rgb et décimal

p { color: rgba(255,0,0,0.5) } /* 0.5 opacity, semi-transparent */
